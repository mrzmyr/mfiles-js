# mfiles-js

> An unoffical M-Files JavaScript Wrapper with promise support

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i mfiles-js --save-dev
```

## Usage

```js
var mfiles-js = require('mfiles-js');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
